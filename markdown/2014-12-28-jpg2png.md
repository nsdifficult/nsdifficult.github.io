##sips介绍      
sips命令介绍可使用man sips命令查看。这里只记录我使用的其中转换图片格式功能的两个笔记。   
##使用
###转换单个jpg到png<!--more-->    

```
sips -s format png 1.jpg --out 1.png
```
如果要转换到其他格式，请将format后的png格式改成其他，如gif。
###批量转换jpg到png    

```
for i in *.jpg;do sips -s format png $i --out png/$i.png;done
```
